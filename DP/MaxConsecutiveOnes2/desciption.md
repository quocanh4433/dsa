
## 487. Max Consecutive Ones II (Leetcode premium and Neetocde)

Given a binary array nums, return the maximum number of consecutive 1s in the array if you can flip at most one 0.

### example 1

Input: nums = [1,0,1,1,0]

Output: 4

Explanation: 

Flip the first zero, the maximum number of consecutive 1s is 4.

### example 2

Input: nums = [1,0,1,1,0,1]

Output: 4

Explanation:

Flip the second zero, the maximum number of consecutive 1s is 4.


### Constraints:

- 1 <= nums.length <= 10^5

- nums[i] is either 0 or 1
